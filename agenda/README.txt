¡¡Este programa solo está disponible en LINUX!!

La agenda está creada con python utilizando las librerias
json
os

La libreria json la utilizaremos para hacer una base de datos que nos sirva para almacenar un nombre y un telefono
Mientras que la librerias os, solo la utilizaremos para borrar la terminal y hacer un menú funcional, en este caso he utilizado 
el comando os.system("clear"), que en linux sirve para limpiar la terminal, mientras que en WINDOWS el comando es "cls", por eso dará error en WINDOWS

Para utilizar este programa habrá que descargarse también el archivo agenda.json (este tiene 3 ejemplos, pero se pueden borrar), lo importante es que el 
archivo, aunque sea uno diferente tenga una "array" dentro de él, para que lo pueda leer como json
